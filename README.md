# astrbot_plugin_qq_keyword_alert

AstrBot 插件。用于监听指定 QQ 群消息，命中关键词后向指定 QQ 私聊发送提醒。

本插件由 **OpenClaw** 配合 **GPT-5.4 Codex** 开发完成。

## 功能

- 按群独立维护关键词、过滤词、开关、正则模式
- 命中后私聊通知指定 QQ
- 简单重复消息冷却
- 管理命令仅限 **AstrBot 管理员私聊** 使用
- 支持按群独立规则管理

## 配置结构

插件使用结构化规则：

```json
{
  "rules": {
    "<群号>": {
      "keywords": ["<关键词1>", "<关键词2>"],
      "exclude_keywords": ["<过滤词1>"],
      "enabled": true,
      "regex_mode": false
    }
  },
  "notify_user_id": "<QQ号>",
  "cooldown_seconds": 30,
  "case_sensitive": false
}
```

## QQ 命令

> 以下命令仅限 **AstrBot 管理员私聊** 使用。

- `/添加群 <群号>`
- `/删除群 <群号>`
- `/开启监控 <群号>`
- `/关闭监控 <群号>`
- `/添加 <群号> <关键词1,关键词2>`
- `/删除 <群号> <关键词>`
- `/添加过滤 <群号> <过滤词1,过滤词2>`
- `/删除过滤 <群号> <过滤词>`
- `/正则模式 <群号> 开|关`
- `/设置通知 <QQ号>`
- `/测试通知`
- `/状态`
- `/关键词列表`
- `/关键词帮助`

## 说明

仓库默认不包含具体业务词、群号、通知目标等隐私配置。请在 AstrBot WebUI 或私聊命令中按需填写你自己的配置。


## 行为说明

- 添加关键词或过滤词前，需先使用 `/添加群 <群号>` 建立该群规则
- `/测试通知` 会明确返回成功或失败
- 过滤日志默认只记录最小必要信息，不直接落完整消息原文
